3 Feature Routing — Which Provider & Model runs each feature?
For each app feature, choose the provider to use (or leave as "Active Provider" to inherit the global default). Then pick or type the model — click the field to see suggestions, or type any model name manually.

Video Interview requires a 👁 vision-capable model (e.g. GPT-4o, Gemini 2.0 Flash, Claude Sonnet 4.5, Llama 3.2 Vision) — it must be able to see the camera feed. Text-only models will not work for video mode.
